## This package contains:
 - Mocks
    - Typeorm
    - Typeorm extension
 - Helpers
    - Launchers (with/without db)
 - Services
    - CRUD
    - Log
    - Microservice metadata
    - Remote config
 - Class validator extra validators
 - Test helpers

See [microservices](https://github.com/Lomray-Software/microservices) for example of usage.
